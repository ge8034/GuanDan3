# 关丹3 语音通话系统

基于 WebRTC 的低延迟 P2P 语音通话系统，支持多人实时语音通信。

## 功能特性

### 核心功能
- ✅ 创建房间并生成唯一房间ID
- ✅ 支持多人同时在线（最多50人）
- ✅ 实时显示当前在线成员列表（用户名、在线状态）
- ✅ 一键发起/结束语音通话
- ✅ 采用 WebRTC 实现低延迟 P2P 语音传输
- ✅ 静音/取消静音控制
- ✅ 音量调节
- ✅ 通话结束后自动释放资源并清空房间数据

### 技术特点
- 🚀 低延迟（目标≤300ms）
- 🔒 端到端加密（WebRTC内置）
- 📱 跨平台支持（桌面浏览器 + 移动端）
- 🎯 无需插件，纯浏览器实现
- ⚡ 高性能，支持50人并发

## 快速开始

### 环境要求
- Node.js v16.0.0 或更高版本
- 现代浏览器（Chrome 90+、Edge 90+、Firefox 88+）
- HTTPS（生产环境必需）

### 安装步骤

#### 1. 安装依赖
```bash
cd voice-chat/server
npm install
```

#### 2. 启动信令服务器
```bash
npm start
```

服务器将在 `ws://localhost:8080` 启动。

#### 3. 打开前端页面
在浏览器中打开 `client/index.html` 文件。

### 使用方法

#### 创建房间
1. 输入用户名
2. 点击"创建房间"按钮
3. 复制生成的房间ID分享给好友

#### 加入房间
1. 输入用户名
2. 输入房间ID
3. 点击"加入房间"按钮

#### 语音通话控制
- 🎤 **静音/取消静音**: 点击麦克风按钮
- 🔊 **音量调节**: 拖动音量滑块
- 📞 **结束通话**: 点击电话按钮

## 故障排查

### 常见问题

#### 1. 无法连接到服务器
- 检查服务器是否启动
- 检查防火墙设置
- 确认端口8080未被占用

#### 2. 麦克风无法使用
- 检查浏览器权限设置
- 确认系统麦克风未被其他应用占用
- 尝试刷新页面重新授权

#### 3. 语音延迟过高
- 检查网络连接质量
- 尝试使用有线网络
- 关闭其他占用带宽的应用

#### 4. 移动端无法使用
- 确保使用HTTPS（WebRTC要求）
- 检查浏览器版本
- 尝试切换Wi-Fi/4G网络
