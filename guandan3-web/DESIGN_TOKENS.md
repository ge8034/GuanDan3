# 富春山居图主题设计令牌文档

## 概述

本文档定义了富春山居图主题的设计系统，包括颜色、排版、间距、圆角、阴影、动画等设计令牌。所有令牌都遵循WCAG 2.1无障碍标准，确保对比度≥4.5:1。

## 设计原则

1. **青绿山水基调**：以中国传统山水画为灵感，采用青绿色系为主色调
2. **水墨意境**：融入水墨画的淡雅风格，使用低饱和度色彩
3. **现代简约**：结合现代UI设计原则，保持界面简洁清晰
4. **无障碍优先**：确保所有交互元素符合WCAG 2.1标准

## 最佳实践

1. **颜色使用**
   - 优先使用主题色，避免自定义颜色
   - 确保文本对比度符合WCAG标准
   - 使用语义化的颜色名称

2. **排版使用**
   - 标题使用semibold（600）字重
   - 正文使用normal（400）字重
   - 保持一致的行高（1.6）

3. **间距使用**
   - 使用8px网格系统
   - 保持一致的间距节奏
   - 避免使用奇数间距

4. **圆角使用**
   - 按钮、卡片使用8-12px圆角
   - 保持一致的圆角风格
   - 避免过度使用圆角

5. **动画使用**
   - 水波纹效果使用300ms时长
   - 使用适当的缓动函数
   - 避免过度动画

6. **图标使用**
   - 优先使用手绘山水图标
   - 保持一致的图标尺寸
   - 为图标提供适当的间距
