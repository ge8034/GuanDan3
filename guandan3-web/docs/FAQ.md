# 掼蛋3 常见问题解答

## 账号相关

### 注册与登录

**Q: 如何注册账号？**

A: 访问官网 guandan3.com，点击右上角"注册"按钮，填写用户名、邮箱和密码，验证邮箱后即可完成注册。

**Q: 忘记密码怎么办？**

A: 点击登录页面的"忘记密码"链接，输入注册邮箱，系统会发送重置密码的链接到您的邮箱。

**Q: 可以修改用户名吗？**

A: 目前不支持修改用户名，建议在注册时仔细选择。

**Q: 如何绑定手机号？**

A: 登录后进入"设置" -> "账号设置"，点击"绑定手机号"，按照提示完成绑定。

**Q: 账号被盗怎么办？**

A: 立即联系客服（support@guandan3.com），提供账号信息和身份证明，我们会协助您找回账号。

## 游戏规则

### 基础规则

**Q: 掼蛋的基本规则是什么？**

A: 掼蛋是四人扑克游戏，使用两副牌（108张），分为两队对抗。每人27张牌，先出完牌的一方获胜。

**Q: 什么是逢人配？**

A: 红桃级牌可以当作任意牌使用，这是掼蛋的特色规则，增加了游戏的策略性。

**Q: 炸弹如何比较大小？**

A: 王炸 > 六张炸弹 > 五张炸弹 > 四张炸弹。相同张数的炸弹比较点数大小。

**Q: 顺子可以包含2和王吗？**

A: 不可以。顺子不能包含2和王，只能使用3-A的牌。

**Q: 连对最少需要几对？**

A: 连对最少需要三对，且点数必须连续。

### 升级规则

**Q: 如何升级？**

A: 获胜方根据对方剩余牌数升级：
- 对方未出牌：升3级
- 对方剩余1-5张：升2级
- 对方剩余6-10张：升1级

**Q: 什么时候游戏结束？**

A: 当一方先升到A级时，游戏结束，该方获胜。

**Q: 级牌是什么？**

A: 级牌是当前回合的等级对应的牌。例如2级时，2就是级牌；A級时，A就是级牌。

## 游戏操作

### 出牌操作

**Q: 如何出牌？**

A: 点击要出的牌选中（选中的牌会向上移动），然后点击"出牌"按钮。

**Q: 可以撤销出牌吗？**

A: 出牌后无法撤销，请仔细选择要出的牌。

**Q: 提示功能如何使用？**

A: 点击"提示"按钮，系统会自动选择可以出的牌，帮助您决策。

**Q: 如何快速选择手牌？**

A: 点击"排序"按钮，手牌会按照点数从大到小排列，方便查找。

**Q: 可以同时出多张牌吗？**

A: 可以，选中多张牌后点击"出牌"按钮，系统会自动识别牌型。

### 游戏设置

**Q: 如何调整音量？**

A: 进入"设置" -> "游戏设置"，拖动音量滑块调整音效和音乐音量。

**Q: 如何切换主题？**

A: 进入"设置" -> "显示设置"，选择"浅色"或"深色"主题。

**Q: 可以自定义牌面样式吗？**

A: 可以，在"设置" -> "显示设置"中选择不同的牌面样式。

**Q: 如何关闭出牌动画？**

A: 进入"设置" -> "游戏设置"，关闭"动画"选项。

## 房间系统

### 房间创建

**Q: 如何创建房间？**

A: 在游戏大厅点击"创建房间"按钮，设置房间名称和密码（可选），选择游戏模式后创建。

**Q: 房间密码可以修改吗？**

A: 创建房间后可以修改密码，进入房间设置即可更改。

**Q: 可以设置AI难度吗？**

A: 可以，创建房间时选择AI难度（简单/中等/困难）。

**Q: 房间最多支持多少人？**

A: 标准房间支持4人，观战模式支持额外2人观战。

### 房间加入

**Q: 如何加入房间？**

A: 在房间列表中选择房间，点击"加入"按钮。如果有密码，需要输入密码。

**Q: 可以加入已满的房间吗？**

A: 不可以，需要等待有空位或创建新房间。

**Q: 如何快速匹配游戏？**

A: 点击"快速开始"按钮，系统会自动为您匹配合适的房间。

**Q: 可以邀请好友加入房间吗？**

A: 可以，在房间中点击"邀请好友"，选择好友发送邀请。

## 好友系统

### 好友管理

**Q: 如何添加好友？**

A: 点击好友列表中的"添加好友"，输入对方用户名发送好友请求。

**Q: 如何删除好友？**

A: 在好友列表中点击好友头像，选择"删除好友"。

**Q: 好友请求在哪里查看？**

A: 好友请求会显示在好友列表的"请求"标签中。

**Q: 可以设置好友分组吗？**

A: 目前不支持好友分组，未来版本可能会添加此功能。

### 好友互动

**Q: 如何邀请好友游戏？**

A: 在好友列表中选择好友，点击"邀请游戏"，好友收到邀请后可以加入。

**Q: 可以和好友聊天吗？**

A: 可以，点击好友头像打开聊天窗口，发送消息。

**Q: 如何查看好友在线状态？**

A: 好友列表中会显示好友的在线状态（在线/离线/游戏中）。

**Q: 可以查看好友的游戏记录吗？**

A: 可以，点击好友头像查看个人资料，包括游戏记录和统计数据。

## 技术问题

### 性能问题

**Q: 游戏卡顿怎么办？**

A: 检查网络连接，关闭其他占用带宽的应用，或尝试刷新页面。如果问题持续，请联系客服。

**Q: 游戏加载很慢怎么办？**

A: 检查网络速度，清除浏览器缓存，或尝试使用其他浏览器。

**Q: 手机上玩游戏发热严重怎么办？**

A: 降低游戏画质，关闭动画效果，或使用散热器。

**Q: 游戏闪退怎么办？**

A: 更新浏览器到最新版本，清除浏览器缓存，或尝试使用其他浏览器。

### 兼容性问题

**Q: 支持哪些浏览器？**

A: 推荐使用Chrome、Safari、Edge等现代浏览器。不支持IE浏览器。

**Q: 支持手机吗？**

A: 支持，可以在手机浏览器中访问 guandan3.com 进行游戏。

**Q: 支持平板吗？**

A: 支持，平板浏览器体验更佳。

**Q: 可以在电视上玩吗？**

A: 如果电视支持浏览器应用，可以访问网站进行游戏。

### 数据问题

**Q: 游戏数据会丢失吗？**

A: 不会，所有游戏数据都保存在云端，登录后即可恢复。

**Q: 如何导出游戏记录？**

A: 目前不支持导出，未来版本可能会添加此功能。

**Q: 可以删除游戏记录吗？**

A: 不可以，游戏记录是系统自动保存的，无法删除。

**Q: 数据会同步到其他设备吗？**

A: 会，登录同一账号，数据会自动同步到所有设备。
