3. 某些边界情况下传递了undefined或null

**建议修复**:
```typescript
// 在AI决策函数入口添加参数校验

  if (!cards || !Array.isArray(cards)) {
    console.error('Invalid cards parameter:', cards);
    return { type: 'pass' };
  }
  // 继续处理...
}
```
