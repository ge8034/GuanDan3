## 🎨 设计规范总结

### Poker主题色彩系统

**主色调**:
```css
/* 牌桌绿色 */
--poker-table: #1a472a
--poker-table-dark: #0d2818
--poker-table-light: #1e5634
--poker-table-border: #2d5a3d

/* 装饰金色 */
--accent-gold: #d4af37
--accent-goldLight: #e5c158
```

**使用规则**:
1. **主要操作** - 深绿渐变背景 + 绿色边框
2. **次要操作** - 中绿渐变背景 + 绿色边框
3. **轮廓样式** - 透明背景 + 金色边框
4. **高亮强调** - 金色文字/边框/图标
5. **状态指示** - 使用Poker主题色（success/error/warning/info）

### 组件样式规范

**所有组件必须遵循**:
- ✅ 使用2px边框（而非1px）
- ✅ 主要边框使用`--poker-table-border`
- ✅ 高亮边框使用`--accent-gold`
- ✅ 背景使用渐变（而非纯色）
- ✅ 添加适当的阴影增强层次感
- ✅ hover状态有明显的视觉反馈
- ✅ 焦点状态使用金色高亮
