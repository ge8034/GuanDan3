5复制连接字符串

点击 Copy 按钮，你会得到类似这样的字符串：

连接字符串格式：
postgresql://postgres.rzzywltxlfgucngfiznx:[显示实际密码]@db.rzzywltxlfgucngfiznx.supabase.co:5432/postgres
↑ ↑ ↑ ↑ ↑
用户名 ← 密码（需要这个！） 主机 端口 数据库
⚠️ 重要提示
  • 复制的连接字符串中，@ 符号前面的部分就是数据库密码
  • 密码通常包含字母、数字和特殊字符，比较长
  • 复制完整的连接字符串，不要截断
6粘贴给我

将完整的连接字符串粘贴给我，格式如下：

postgresql://postgres.rzzywltxlfgucngfiznx:你的实际密码@db.rzzywltxlfgucngfiznx.supabase.co:5432/postgres
⚠️ 安全警告
  • 数据库密码具有完全访问权限
  • 绝对不要提交到 Git 仓库
  • 绝对不要在前端代码中使用
  • 只会在 .env.local 文件中使用
